## 📌 Project Overview
ScreenSense AI is a machine learning project that analyzes behavioral usage patterns—such as daily screen time, notification frequency, and sleep hours to predict if a user is at risk of smartphone addiction.

## 📊 Insights from Data
Using Exploratory Data Analysis (EDA), we identified several key trends:

- **Gaming Distribution:** High-intensity gaming (4+ hours/day) is most heavily concentrated in the **18–25 age group**. The distribution shows a significant "long tail" for users in this bracket, identifying them as a primary demographic for high-usage patterns.
- **Social Media Trends:** Average social media engagement peaks significantly within the **18–30 demographic**. Data shows that high engagement in this category serves as a primary indicator for addiction risk.
- **Behavioral Correlation:** Analysis reveals a strong positive correlation between **Notification Count** and **Total Screen Time**, suggesting that frequent device "pings" are a major driver of increased daily usage.
